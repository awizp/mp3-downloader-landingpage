# Tamil MP3 Downloader - Download More than 1K+ songs in single click | Landing page

A clean and responsive landing page built using HTML, CSS and Bootstrap for a Tamil MP3 songs downloader application. Designed with a modern layout, fast loading and mobile friendly.

### Features

* Dynamic Component Rendering: Uses Bootstrap to fetch and render components into the landing page.

* Responsive Design: The layout adapts to different devices, ensuring a great user experience on desktop, tablet, and mobile devices.

### Languages used

* HTML: The structure of the web page.

* CSS: Used for styling and media queries to make the design responsive.

* Bootstrap: To fetch and dynamically insert component into the page to clean look.

### Installation

1. Clone this repository to your local machine

``````git clone https://github.com/awizp/mp3-downloader-landingpage.git``````

2. Open the ```index.html``` file in your browser to view the project.
